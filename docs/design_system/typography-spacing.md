# CardToo Typography and Spacing Guideline

## Mobile Only Focus
Sistem ini fokus ke layar hp dengan 8pt grid.

## Typography
- Heading 1: 24px (1.5rem) / Bold
- Heading 2: 20px (1.25rem) / Semibold
- Body Large: 16px (1rem) / Medium
- Body Normal: 14px (0.875rem) / Regular
- Body Small: 12px (0.75rem) / Regular

## Spacing System
- Space XS: 4px
- Space Small: 8px
- Space Medium: 16px (Safe Padding)
- Space Large: 24px
- Space XL: 32px

## Radius (Modern UI)
- Radius Card Large: 24px (Layar penuh / Modal / Sheet)
- Radius Card Standard: 16px (Card produk / Item list)
- Radius Button & Input: 24px - 26px (Pill shape untuk kesan modern & native feel)
- Radius Badge/Tag: 8px - 12px
