# CardToo Color System

* Primary: [#00CAE0] (Warna utama brand; tombol utama; link aktif)
* Secondary: [#A78BFA] (Warna pendukung; tombol sekunder)
* Accent: [#434343] (Warna highlight; badge; ikon)

* Border: [#E5E7EB] (Garis pemisah; outline)
* Text Primary: [#111827] (Teks utama; judul tebal)
* Text Secondary: [#6B7280] (Teks pendukung; paragraf biasa)
